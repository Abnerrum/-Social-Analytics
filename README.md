# 📊 Social Analytics Dashboard

Um dashboard interativo para análise de redes sociais, mostrando métricas como:

- Total de seguidores
- Impressões
- Taxa de engajamento
- Crescimento de seguidores
- Cliques em links

## 🚀 Tecnologias usadas

- HTML5
- CSS3
- JavaScript
- Chart.js

## 📈 Funcionalidades

- Alternar período (7 dias, 30 dias, 90 dias)
- Gráficos de engajamento (likes, comentários e compartilhamentos)
- Distribuição de público por plataforma
- Alcance por tipo de conteúdo
- Tabela com posts mais relevantes
